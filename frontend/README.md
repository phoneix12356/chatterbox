# ChatterBox Frontend

This is the React frontend for the ChatterBox application.

## Prerequisites
- Node.js (v16 or higher)
- NPM

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open your browser at `http://localhost:5173`.

## Features
- **Authentication**: Login and Signup with JWT integration.
- **Home Feed**: View posts and create new ones.
- **Notifications**: Real-time notifications using WebSockets.
- **Profile**: User profile with stats and posts grid.

## Tech Stack
- React (Vite)
- Redux Toolkit
- Tailwind CSS
- Axios
- React Router DOM
- Lucide React (Icons)
